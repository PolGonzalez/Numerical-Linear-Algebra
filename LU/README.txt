README

1. Les funcions principals "lu", "resol" i "inversa" es troben al fitxer "funcions.cc"
2. Les altres funcions que no es demanen però que s'utilitzen al main i a les funcions anteriors es troben al fitxer "matlib.cc"
3. S'inclouen els arxius "funcions.h" i "matlib.h" per al correcte funcionament del programa

Per tant, per l'execució del programa: 

        - Compilar: g++ -Wall main.cc matlib.cc funcions.cc
        - Executar: ./provaLU "DadesMatriu.dat"
